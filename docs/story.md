# Grundidee
- Insel wurde von Architekt erschaffen
- Core Compiler ist zerbrochen und in Stücke verstreut
- Insel muss programmiert werden, um den Core Compiler wiederherzustellen
- 1 Level = 1 Konzept der Java Programmierung

# Level 1: Variablen und Datentypen
int, double, boolean, string

## Beispiele
- Deklaration und Initialisierung
- Speichern von Daten in Variablen
- Manipulation von bereits bestehenden Variablen

# Level 2: if-Verzweigung
&&, ||, ==, >=, <=, <, >, !=, ...

## Beispiele
- Zerstörte Brücke; wenn Bedingung korrekt: Brücke repariert sich
- Wenn Wachen schlafen und Schlüssel vorhanden: öffne Tor

# Level 3: Schleifen
for, while

## Beispiele
- Durchsuche x Objekte
- Reibe Holz aneinander, bis das Feuer brennt

# Level 4: Methoden
void, return

## Beispiele
- Methode zum Fackel anzünden

# Level 5: Objekte
new, this, set, get

## Beispiele
- Baue Soldaten
- Erstelle 5 Soldaten, um Monster zu besiegen
- Setze Angriffsschaden der Soldaten
- Hole Lebenspunkte der Monster nach Angriff

## Level 6: Vererbung und Polymorphie
## Level 7: Interfaces und abstrakte Klassen

# Finales Level: Exceptions
try, catch, finally

## Beispiele
- Core Compiler hat Fehlfunktion, muss behandelt werden

## Mögliche Story Twists
- Es stellt sich raus, dass ein höheres Wesen den Core Compiler versucht zu zerstören
- Man muss das höhere Wesen indirekt bekämpfen, indem man die geworfenen Exceptions behandelt
- Gleichzeitig greifen Monster an und man muss diese mit selbst erstellten Soldaten abwehren
