# 1. Ziel der Woche
Ziel dieser Woche war es, die inhaltliche Ausarbeitung der ersten Kapitel weiter voranzutreiben, die Codeüberprüfung für Aufgaben zu ermöglichen und erste visuelle Elemente für das Spiel zu erstellen.

# 2. Umgesetzte Aufgaben
## Technische Umsetzung
- Erweiterung der bestehenden Datenbankstruktur
- Integration einer Library zur Ausführung von durch den Spieler geschriebenem Code
- Implementierung eines Vergleichs zwischen Programmausgabe und erwarteter Lösung zur Aufgabenüberprüfung

## Story / Game Design
- Konzeption von Kapitel 1 (Variablen)
- Konzeption von Kapitel 2 (If-Verzweigungen)
- Erstellung des Skripts für Kapitel 3 (Schleifen)

## Visuelle Gestaltung
- Erstellung erster Bilder für die Spielatmosphäre

# 3. Fortschritt im Projekt
Diese Woche wurde die inhaltliche Struktur der ersten Lernkapitel weiter ausgearbeitet. Die Kapitel 1 und 2 wurden vollständig konzipiert, während für Kapitel 3 bereits ein Skript erstellt wurde.
Technisch wurde eine Library integriert, die es ermöglicht, den vom Spieler geschriebenen Code auszuführen und dessen Ausgabe automatisch mit einer erwarteten Lösung zu vergleichen. Dadurch können Programmieraufgaben im Spiel überprüft werden.
Zusätzlich wurden erste visuelle Elemente erstellt, um die Atmosphäre der Spielwelt zu unterstützen.

# 4. Probleme & Herausforderungen
- Integration der Library zur sicheren Ausführung von nutzereigenem Code
- Erweiterung der Datenbankstruktur ohne bestehende Funktionen zu beeinträchtigen
- Abstimmung zwischen Spielmechanik und didaktischem Aufbau der Lernkapitel
